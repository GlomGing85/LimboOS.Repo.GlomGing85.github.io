WHAT IS LIMBOOS

A lightweight OS
for the Limbo Emulator

LimboOS is a minimal, fast and beautiful operating system built specifically to run inside the Limbo PC Emulator (x86). Based on Alpine Linux with a custom graphical shell.

Powered by
Alpine Linux
x86
Live CD

Built for simplicity

Everything you need, nothing you don't.

Lightning Fast

Boots in seconds. Extremely low resource usage — perfect for emulation.

Custom Shell

Beautiful graphical interface built from scratch with keyboard-first design.

Built-in Apps

Terminal, File Manager, Settings, and growing collection of lightweight apps.
